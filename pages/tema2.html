Tema 2: Huruf Berbaris

Belajar Fathah, Kasrah dan Dammah

بَ

BA Fathah

FATHAH

  • Baris atas ( بَ )
  • Bunyi: 'a' pendek
  • Contoh: بَ (ba), تَ (ta), ثَ (tha)
  • Dibaca dari atas ke bawah

بِ

BA Kasrah

KASRAH

  • Baris bawah ( بِ )
  • Bunyi: 'i' pendek
  • Contoh: بِ (bi), تِ (ti), ثِ (thi)
  • Dibaca dari bawah ke atas

بُ

BA Dammah

DAMMAH

  • Baris depan ( بُ )
  • Bunyi: 'u' pendek
  • Contoh: بُ (bu), تُ (tu), ثُ (thu)
  • Dibaca dengan memuncungkan mulut

بْ

BA Sukun

SUKUN

  • Tanda bulat ( بْ )
  • Tidak ada bunyi vokal
  • Contoh: بْ (b), تْ (t), ثْ (th)
  • Huruf mati tanpa baris

بً

BA Fathatain

TANWIN FATHAH

  • Dua baris atas ( بً )
  • Bunyi: 'an'
  • Contoh: بً (ban), تً (tan), ثً (than)
  • Biasanya di akhir kata

بٍ

BA Kasratain

TANWIN KASRAH

  • Dua baris bawah ( بٍ )
  • Bunyi: 'in'
  • Contoh: بٍ (bin), تٍ (tin), ثٍ (thin)
  • Biasanya di akhir kata

بٌ

BA Dammatain

TANWIN DAMMAH

  • Dua baris depan ( >بٌ )
  • Bunyi: 'un'
  • Contoh: بٌ (bun), تٌ (tun), ثٌ (thun)
  • Biasanya di akhir kata
